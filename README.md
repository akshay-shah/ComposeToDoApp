# ComposeToDoApp

Jetpack Compose App

# Stack

- Jetpack Compose
- MVVM
- Coroutine Flows
- Room Database
- Dagger Hilt
- Preference DataStore
- Compose Animations
- Dark Mode Support

# Features

- Splash Screen
- **Add** new task
- **Filter** tasks by priority
- **Update** tasks
- **Delete** all tasks.
- **Search** tasks in complete database
- **Undo** deleted task
- **Swipe to delete** task
